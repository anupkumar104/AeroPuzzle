AeroPuzzle 🧩

AeroPuzzle is an AI-powered gesture-controlled puzzle game that combines Computer Vision and interactive gaming. Users can select an image area using hand gestures, generate a puzzle instantly, and solve it using pinch-and-drag interactions without touching the keyboard or mouse.
